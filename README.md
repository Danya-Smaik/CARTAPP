# ElectroHub

An e-commerce website for electronics built with HTML, CSS, and JavaScript.  
The project provides a modern shopping experience with product search, favorites, shopping cart management, and user authentication using browser storage.

## 🌐 Live Demo

https://cart-proj-js.netlify.app/

## ✨ Features

-  Search products by name and filter them by category.
-  Add products to the shopping cart with quantity controls (+/-) and automatic total price calculation.
-  Save favorite products with data synchronized across pages using Local Storage.
-  User registration and login with authentication handled using Local Storage and Session Storage.
-  Custom brand identity, including logo, color palette, and typography, designed and refined from scratch.
-  Responsive design for a smooth experience across different screen sizes.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- Session Storage
- Font Awesome
- Google Fonts
